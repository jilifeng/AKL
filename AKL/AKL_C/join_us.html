    7、有丰富的网络推广执行经验,熟悉网络店铺推广的方式方法,擅长网络/社区交流、熟悉网站 及网店操作;诸如网站联盟、广告投放、论坛营销等一切有效的推广方式,能独立策划店铺促销活动方案


    任职资格：

    1、教育背景：电子商务或营销类相关专业

    2、工作经验：3年以上工作经验，2年以上电子商务网店运营经验，有电子产品类电子商务运营经验优先；

    3、知识技能：

    a) 精通B2C网络销售运营业务流程；

    b) 精通网络推广和网络营销的策划及执行；

    c) 熟悉B2C网站的技术模块构成和建设应用；

    4、个人素质：有较强的逻辑思维推理能力、沟通协调能力及较强的团队合作意识；

    5、有较强的工作投入意识，有钻研和创新及学习的能力

    IT经理 1名     2015-8-9

    岗位职责

    1、拟定和执行企业信息化战略

    负责制订公司信息化中长期战略规划、当年滚动实施计划，制定企业信息化管理制度、制定信息化标准规范；负责公司信息化网络规划、建设组织、制订IT基础资源（硬、软件）运行流程、制定网络安全、信息安全措施并组织实施，实现IT资源集约管理；负责公司集成信息系统总体构架，构建企业信息化实施组织，结合业务流程重组、项目管理实施企业集成信息系统。负责公司及各分公司网站建设及总体规划。

    2、办公自动化系统开发与运行

    根据公司发展战略和实际需要，组织实施公司办公自动化系统、网站的运行管理和维护与更新，协助信息管理工作；负责公司办公自动化设备（计算机及其软件、打印机）的维护、管理工作。

    3、企业信息资源开发

    根据企业发展战略和信息化战略要求，负责企业内外部信息资源开发利用。导入知识管理，组织建立企业产业政策信息资源、竞争对手信息资源、供应商信息资源、企业客户信息资源、企业基础数据资源五大信息资源库。

    4、建立信息化评价体系

    根据公司信息化战略和企业实情，建立公司信息化评价体系和执行标准、制定全员信息化培训计划。

    5、信息处理

    负责信息的收集、汇总、分析研究，定期编写信息分析报告报公司领导决策参考；参与公司专用管理标准和制度的制定和修改。

    6、 ERP等信息平台的开发及实施

    负责公司计算机开发应用计划，有步骤地开发使用应用管理软件，逐步实现企业管理现代化、信息化；负责公司ERP系统项目的论证、引进（或开发）与实施，组织ERP系统与企业状况之间关系的分析，确保公司ERP系统的顺利运行。

    7、控制信息设备预算

    负责控制部门预算，降低费用成本，组织公司计算机相关设备的维护、添置、验收、发放登记归档，以及管理软件的咨询、设计、采购、测试、验收、日常维护，并提出可行性方案等工作。

    8、协助其它部门管理

    协助其它部门实施CAD、PDM、CAPP等项目信息管理，协同其它管理部门实施设备管理、人事管理、客户关系管理等信息化管理。

    9、日常管理

    主持本部门日常全面工作，编制本部门年、月工作计划及资金计划，总结年、月度工作；负责本部门员工考核。


    任职资格

    1、计算机、电子、信息相关专业大学以上学历

    2、五年以上软件开发工作经验，至少完成过三个大型的软件开发项目

    3、精通WinForm、C#、XML等软件开发语言以及多线程设计方法，精通Windows NT平台下的C/S三层结构软件开发以及Microsoft Visual Studio 2005环境和VSS的使用

    4、对大型数据库SQL SERVER 2000/2005、MYSQL、Oracle、Sybase等能熟练掌握，并能进行数据库编程和优化管理

    5、熟悉软件开发过程以及设计模式

    6、具有很强的文档撰写能力，独立分析和解决问题的能力

    7、三年以上团队管理经验，优秀的沟通能力和团队建设能力
